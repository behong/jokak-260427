# 지조각 유튜브 롱폼 업로드 공통 지침

이 문서는 롱폼 영상 제작, 업로드 준비, 예약 공개, 공개 후 확인 작업에만 적용한다. 숏폼, 네이버 클립, 일반 대시보드 개발에는 적용하지 않는다.

## 적용 우선순위

1. `docs/longform-upload-jobs/` 아래의 해당 영상별 지침
2. 이 문서의 롱폼 공통 지침
3. 앱의 기존 기본값

영상별 지침과 공통 지침이 다르면 영상별 지침을 우선한다.

## 메타데이터

- 제목 후보는 3개를 준비한다.
- 구분자는 전각 세로 막대 `｜`를 사용하고 양쪽에 공백을 한 칸씩 둔다.
- 영상 내용과 관계없는 검색어와 과도한 자극 표현은 사용하지 않는다.
- 영상별 지침에 설명이 지정되어 있으면 문구를 임의로 고치지 않는다.
- 태그는 영상별 지침에 지정된 순서대로 쉼표로 구분한다.

## 썸네일

- Codex는 롱폼 썸네일을 생성하지 않는다.
- 사용자가 별도로 제공한 16:9 이미지 3장만 사용한다.
- 제공 이미지를 수정, 보정, 크롭하거나 자동 생성 이미지로 교체하지 않는다.
- 세 장을 지정된 순서대로 YouTube A/B 테스트에 등록한다.

## 기본 업로드 설정

- 시청자층: 아동용 아님
- 연령 제한: 없음
- 유료 프로모션: 실제 포함된 경우에만 표시
- 공동작업자: 없음
- 동영상 언어: 한국어
- 제목 및 설명 언어: 한국어
- 자막 인증: 해당 없음
- 카테고리: 인물/블로그
- 라이선스: 표준 YouTube 라이선스
- 퍼가기: 허용
- 구독 피드 게시 및 구독자 알림: 켜기
- Shorts 리믹스: 권리 문제가 없으면 허용
- 녹화 날짜와 위치: 생략 가능
- 자동 장소: 끄기
- 자동 챕터 및 주요 순간: 켜기
- 댓글: 모든 댓글 허용
- 검토 방식: 부적절할 수 있는 댓글 검토 보류
- 댓글 정렬: 인기 댓글
- 좋아요 수 표시: 켜기

## AI 사용 공개

AI 음성, AI 생성 이미지, AI 생성 영상, 실제 촬영처럼 보이는 생성 장면 중 하나라도 사용했다면 AI 사용 항목에서 `예`를 선택한다. 단순 자막, 색 보정, 화면 전환만 사용했다면 `아니요`를 선택한다.

## 자막

- 한국어 자막을 추가한다.
- 원고가 있으면 자막 파일로 변환해 업로드한다.
- 원고가 없으면 자동 자막 생성 후 오탈자를 수정한다.

## 엔딩 화면과 카드

- 마지막 20초에 왼쪽 채널 구독 버튼과 오른쪽 시청자 맞춤 동영상을 배치한다.
- 영상 속 핵심 자막을 가리지 않는다.
- 기존 엔딩 화면 설정이 있으면 재사용한다.
- 관련 힐링 낭독 영상이 있을 때만 9~12분 지점에 카드 1개를 추가한다.

## 공개 후 확인

공개 후 고정 댓글을 등록하고, 24시간 동안 다음 항목을 확인한다.

- 노출수
- 노출 클릭률
- 평균 시청 지속 시간
- 평균 조회율
- 첫 30초 이탈률
- 썸네일 A/B 테스트 결과
- 댓글 반응

성과가 낮으면 썸네일 문구와 대비, 제목 첫 문장, 영상 첫 30초, 설명 첫 두 줄 순서로 검토한다.
